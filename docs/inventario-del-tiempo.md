## 4. CГіmo las decisiones redistribuyen el tiempo

Las decisiones de vida no solo cuestan dinero вҖ” cuestan tiempo. Cada decisiГіn **fuerza una redistribuciГіn** del presupuesto semanal:

### Tener un hijo
```
Antes:  Familia 6h/sem  вҶ’ DespuГ©s: Familia 28-35h/sem (primeros 5 aГұos)
Se resta de: Aventura (-8h), Crecimiento (-6h), Comunidad (-5h), Trabajo (-4h)
Efecto: Las otras riquezas crecen mГЎs lento o decaen temporalmente
```

### Comprar casa lejos del trabajo
```
Antes:  Commute 5h/sem  вҶ’ DespuГ©s: Commute 12-15h/sem
Se resta de: Crecimiento (-3h), Bienestar (-2h), Comunidad (-2h)
Efecto: PГ©rdida silenciosa acumulada de 300-500h/aГұo
```

### Emprender un negocio
```
Antes:  Trabajo 40h/sem вҶ’ DespuГ©s: Trabajo 55-70h/sem (primeros 3-5 aГұos)
Se resta de: Bienestar (-5h), Familia (-5h), Aventura (-5h), Comunidad (-3h)
Efecto: Potencial de retorno alto, pero riesgo de deterioro en otras riquezas
```

### Estudiar un posgrado mientras trabaja
```
Antes:  Crecimiento 10h/sem вҶ’ DespuГ©s: Crecimiento 25h/sem (2-3 aГұos)
Se resta de: Aventura (-5h), Comunidad (-4h), Familia (-3h), Bienestar (-3h)
Efecto: InversiГіn temporal que multiplica ingresos y opciones futuras
```

### Decidir viajar un mes
```
Evento puntual: 4 semanas donde Aventura = 80-90% del tiempo
Se pausa: Trabajo (si no es remoto), Dinero (gasto), rutina de Bienestar
Se gana: Aventura ++, Crecimiento +, Comunidad + (nuevas conexiones)
```
